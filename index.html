Kairos — Screen Time Blocker

Kairos is an iOS screen-time app operated by Alexander Spiridellis (Sole Proprietor, United States). It helps people block distracting apps, set daily limits, and take back their time. Kairos is live on the Apple App Store.

View Kairos on the App Store

Text messaging program

Kairos offers an optional accountability-partner text program: with explicit double opt-in confirmation, a user's chosen partner receives one daily text summarizing which self-set app limits the user exceeded.
